## Problem List

### Question 1
Given \(2^a = 3^b = 6^c\), find the value of \(\frac{ab}{ac + bc}\).

### Question 2
Given \(2^a = 5\), \(8^b = 0.1\), find the value of \((2a + 6b + 1)^{2026}\).

### Question 3
Solve the equation \(x^x = 3^{x+54}\), find \(x\).

### Question 4
Solve the equation \(x^2 + 2\sqrt{6}x - 5 = 0\), find \(x^2\).

### Question 5
Given \(x = \frac{16}{\sqrt{17}-1}\), find the value of \(x^5 + 2x^4 - 17x^3 - x^2 + 17\).

### Question 6
Given \(x^2 - x - 1 = 0\), find the value of \(\sqrt{x^4 + \frac{7}{x^4}}\).

### Question 7
Find the maximum value of \(5\sqrt{x-1} + \sqrt{9-2x}\).

### Question 8
Given \(x^4 + x + 1 = 0\), find the value of \(x^3 - x^2\).

### Question 9
Solve the equation \(x^{x^3} = 3375\), find \(x\).

### Question 10
Given \(12^m = 18\), find the value of \(2^{\frac{2m-1}{m-2}}\).
