这是 Python 趣味数学公选课的期末考试项目，包含 10 个数学问题的 Python 解决方案。

## 题目列表

### 问题 1
已知 \(2^a = k\)，\(3^b = k\)，\(6^c = k\)，求 \(\frac{ab}{ac + bc}\) 的值。

### 问题 2
已知 \(2^a = 5\)，\(8^b = 0.1\)，求 \((2a + 6b + 1)^{2026}\) 的值。

### 问题 3
解方程 \(x \ln x = (x + 54) \ln 3\)。

### 问题 4-9
各类数学问题求解。

### 问题 10
已知 \(12^m = 18\)，求 \(2^{\frac{2m-1}{m-2}}\) 的值。

## 技术栈

- **Python 3.x** - 编程语言
- **sympy** - 符号计算库
- **math** - 数学函数库
- **numpy** - 数值计算库
- **mpmath** - 高精度数学计算库


运行单个问题的解决方案：
