## 题目列表

### Question 1
已知 \(2^a = 3^b = 6^c\)，求 \(\frac{ab}{ac + bc}\) 的值。

### Question 2
已知 \(2^a = 5\)，\(8^b = 0.1\)，求 \((2a + 6b + 1)^{2026}\) 的值。

### Question 3
解方程 \(x^x = 3^{x+54}\)，求 \(x\)。

### Question 4
解方程 \(x^2 + 2\sqrt{6}x - 5 = 0\)，求 \(x^2\)。

### Question 5
已知 \(x = \frac{16}{\sqrt{17}-1}\)，求 \(x^5 + 2x^4 - 17x^3 - x^2 + 17\) 的值。

### Question 6
已知 \(x^2 - x - 1 = 0\)，求 \(\sqrt{x^4 + \frac{7}{x^4}}\) 的值。

### Question 7
求式子 \(5\sqrt{x-1} + \sqrt{9-2x}\) 的最大值。

### Question 8
已知 \(x^4 + x + 1 = 0\)，求 \(x^3 - x^2\) 的值。

### Question 9
解方程 \(x^{x^3} = 3375\)，求 \(x\)。

### Question 10
已知 \(12^m = 18\)，求 \(2^{\frac{2m-1}{m-2}}\) 的值。
